После запуска, чтобы игра началась, необходимо расставить на поле начальное положение
живых клеток. Для этого нажмите на клетки, которые хотите сделать живыми. После создания
начального поля, у вас на выбор будет 3 кнопки:'Evolve', 'Start Reprodusing' и 'Reset World'.
'Evolve' делает одну итерацию "жизни" всех клеток.
'Start Reprodusing' запускает "жизнь" клеток.
'Reset World' очищает поле от "живых" клеток.

Версию, которая запускается прямым открытием файла index.html можно посмотреть здесь https://github.com/Delta145/game_of_life_js/tree/fb1c64c8ad3d33ed2851c47d77d2c31363186f49
Текущая версия может запустится только на сервере или, например, при открытие через IDE.
